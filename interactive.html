High-fidelity Prototype

Streamlining ambulance services seamlessly using autonomous technology is the goal of this project.

As the project’s solution, three more rounds of user testing and iterations were applied on the interactive prototype to ensure that as much feedback could be applied to the final high-fidelity prototype. To ensure that nothing was overlooked, suggestions made in previous iterations were reflected on to ensure that all decisions were user friendly. The option to add a profile picture was created using user feedback, with this opportunity updating phrasing of terms to be consistent with what medical professionals used. Accessibility issues were additionally considered, accommodating an option of a ‘colour-blind mode’ in the settings of the watch.

Conclusion

As the first application interface project undertaken as a student, this was a great learning experience, requiring one to be flexible in their array of skills. This project alone required proficiency in Sketch, Adobe Photoshop, CSS, HTML and video editing (Premier & Adobe After Effects) in order to complete all tasks successfully. Technical expertise aside, numerous research methods were learnt as well as an understanding of the extensive research required to pitch a project.
